I built Stuff Stash, a self-hosted home inventory solution with a native
phone app, because I wanted an inventory system but I didn't have time to maintain it.

Stuff Stash can be maintained by clicking, tapping, and typing, but I use
the LLM-powered voice mode because I value my time.

Stuff Stash supports cloud and local models for text to speech, speech to text, and LLMs.

## Why It Exists

My previous inventory solution was invaluable when I needed to find something 
I had in storage. 

But when I needed to move things around, add new items, or lend something to a friend,
it fell short. 

It wasn't for lack of _functionality_, but rather the friction was too high. 

</ContainerSection>

<ContainerSection>

## Core Tenants

- **Low-friction upkeep:** inventory lifecycle must be manageable with native voice
  control
- **Local control:** the app is designed for self-hosting with separate API and
  web deployables.
- **Native SSO:** authentication uses OIDC from the beginning.
- **Scoped sharing:** tenants and inventories protect household boundaries.
- **Portable data:** import and export are core trust promises.
